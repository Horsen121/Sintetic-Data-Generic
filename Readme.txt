Инструкция по работе:

1) Запустить код через __main__;
2) В открывшемся окне выбрать пункт "Настроки";
3) Загрузить конфигурацию из папки config или использовать (при необходимости можно изменить параметры, а также удалить, добавить или изменить свойства столбцов с помощью ПКМ);
4) После совершённыех манипуляций можно закрыть окно с настройками, предварительно нажав "Сохранить конфигурацию";
5) В основном окне нажать "Создание данных" и ожидать генерации таблци.

Возможные генерации:
"typeGeneration" : "######"

"######":
Numbers	
	"range" : "0-100" (примеры условий генераций, которые можно вписать в json или dataclass)
Strings
	"length" : "100"
Masks
	"mask" : "123####145##"
	"typeMask" : "str" или ["a", "б", "в"]
Data
	"firstBorder" : "YYYYMMDD" (заменить буквы на цифры, например, 20220807)
	"secondBorder" : "YYYYMMDD" 
Timestamp
	"firstBorder" : "0" (вместо первой границы число от нуля до второй границы)
	"secondBorder" : "1659879902"
Collection
	"list" : ["а", "о", "у"] или {"а":2, "о":5, "у":3} (второй вариант для расширенного функционала)
Join
	"table" : "table_name"
	"column" : "column_number" (int)
Name
Lastname
FIO
Patronymic
Email
Phone
